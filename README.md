# ☕ Coffee House — Интернет-магазин кофе

Полнофункциональный интернет-магазин, разработанный в рамках учебного технического задания. Проект представляет собой комплексное решение: от адаптивной верстки до серверной логики и работы с базой данных.

---

## 🚀 Технологический стек

Проект реализован с использованием классического стека веб-разработки:
* **Frontend:** HTML5 (семантическая верстка), CSS3 (Flexbox, Grid, CSS Variables), JavaScript (ES6+).
* **Backend:** PHP (серверная логика, обработка запросов).
* **Database:** MySQL (хранение данных пользователей, товаров и заказов).
* **Interactivity:** AJAX (для работы с корзиной без перезагрузки страницы).

---

## ✨ Реализованный функционал

### 1. Структура и навигация
* **Header:** Логотип, динамическое меню навигации и компактная форма авторизации.
* **Main:** Динамическая смена контента в зависимости от выбранного раздела.
* **Footer:** Контактные данные и информация о правах.
* **Разделы:** Главная (обзор), Магазин (каталог), О нас (информация и обратная связь).

### 2. Магазин и работа с товарами

* **Два режима отображения:** Сетка (карточки) и Таблица (список).
* **Детальная страница:** Увеличенное фото, подробные характеристики (извлекаются из JSON), описание и статус наличия.
* **Корзина (для авторизованных):** * Добавление/удаление товаров.
    * Изменение количества "на лету".
    * Автоматический расчет итоговой суммы.

### 3. Интерактив и UI/UX
* **Слайд-шоу:** Автоматическое переключение баннеров на главной с возможностью ручного управления (стрелки и точки).
* **Анимации:** Плавные эффекты при наведении (hover) и взаимодействии с элементами.
* **Адаптивность:** Полная поддержка мобильных устройств и планшетов.

### 4. Безопасность и Личный кабинет
* **Система регистрации:** Сохранение новых пользователей в БД.
* **Безопасность:** Обязательное хеширование паролей.
* **Обратная связь:** Форма для связи с командой на странице «О нас».

## 🗄️ Структура Базы Данных

Проект использует реляционную базу данных MySQL со следующими таблицами:

| Таблица | Описание |
| :--- | :--- |
| `users` | Хранение данных пользователей и хешированных паролей. |
| `categories` | Категории товаров (зерновой, молотый, аксессуары и т.д.). |
| `products` | Каталог товаров с полем JSON для гибких характеристик. |
| `cart` | Связующая таблица для хранения товаров в корзинах пользователей. |
