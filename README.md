Coffee House - Интернет-магазин кофе
Описание проекта
Coffee House - это полнофункциональный интернет-магазин по продаже кофе, разработанный в соответствии с техническим заданием. Проект демонстрирует навыки веб-разработки с использованием HTML, CSS, JavaScript и PHP.

Реализованный функционал
1. Общая структура сайта
✅ Шапка (header) - содержит логотип и навигационную панель

✅ Основная часть (main) - динамически изменяемое содержимое

✅ Футер (footer) - контактная информация и копирайт

2. Навигация
✅ Навигационная панель с ссылками на 3 страницы:

Главная - информация о магазине и обзор продукции

Магазин - каталог продукции

О нас - информация о компании и команде

3. Дополнительные возможности
✅ Слайд-шоу на главной странице с автоматическим переключением

✅ Ручное управление слайдами (кнопки "вперед/назад" и точки-индикаторы)

4. Работа с продукцией
Режимы просмотра:
✅ Режим сетки - отображение товаров в виде карточек с изображением, ценой и кратким описанием

✅ Режим таблицы - табличное представление всех товаров

Детальная страница товара:
✅ Открывается при клике на товар (через кнопку "Подробнее")

✅ Содержит увеличенное изображение

✅ Характеристики товара (зависят от категории)

✅ Полное описание

✅ Информация о наличии на складе

5. Авторизация и пользователи
✅ Форма авторизации в шапке сайта

✅ Форма регистрации новых пользователей

✅ Хранение пользователей в базе данных

✅ Хеширование паролей

6. Функционал для авторизованных пользователей
✅ Список покупок (корзина):

Добавление товаров

Удаление товаров

Изменение количества

Подсчет общей суммы

✅ Форма обратной связи на странице "О нас"

7. Дизайн и стилизация
✅ Современный адаптивный дизайн

✅ Единая цветовая гамма в кофейных тонах

✅ Правильные отступы и позиционирование элементов

✅ Анимации при наведении

✅ Адаптация под мобильные устройства

8. Техническая реализация
✅ HTML - семантическая верстка

✅ CSS - современные технологии (Flexbox, Grid, переменные)

✅ JavaScript - интерактивность и AJAX-запросы

✅ PHP - серверная логика и работа с базой данных

✅ MySQL - хранение данных (пользователи, товары, корзина)

9. База данных
Таблица пользователей (users)

Таблица категорий (categories)

Таблица товаров (products) с JSON-полем для характеристик

Таблица корзины (cart)

Дополнительные баллы
В проекте реализованы следующие возможности, дающие дополнительные баллы:

✅ Слайд-шоу на главной странице

✅ Регистрация пользователей с сохранением в БД

✅ Современный CSS с использованием переменных, Grid, Flexbox

✅ Адаптивный дизайн для всех устройств

✅ Анимации при взаимодействии с элементами

✅ AJAX-запросы для добавления в корзину без перезагрузки
